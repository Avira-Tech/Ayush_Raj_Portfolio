# Ayush Raj | Multi-Disciplinary Designer Portfolio

A premium, high-performance portfolio website built with **React**, **Vite**, and **Framer Motion**. This project showcases a sophisticated glassmorphism design with interactive 3D elements and a seamless user experience.

## ✨ Features

- **Premium Glassmorphism UI**: High-end aesthetic with frosted glass effects, layered shadows, and radial gradients.
- **Interactive Hero Section**: Features a mouse-following interactive glow, 3D floating visual elements, and a dynamic shimmer headline.
- **Selected Work Showcase**: Dynamic gallery with category filtering (Brochures, Meta Ads, Social Media, etc.) and a "Show More/Less" feature.
- **Professional Journey**: A detailed timeline of professional experience with specialized skill tags and company details.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Functional Contact Form**: Sleek, glass-styled form for professional inquiries.
- **Direct Resume Download**: Integrated CV download buttons in the Hero and Experience sections.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS (Custom Design System)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   ```

2. **Navigate to the project directory**
   ```bash
   cd "Ayush Portfolio"
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```
